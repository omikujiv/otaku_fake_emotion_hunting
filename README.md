* 表情豊かな人間になることをアシストすることが期待されます

# 対応キーワード (コード改変により追加可能)
* 顔ない
* ワロタ
* ワロス
* 草
* わら
* 泣いた
* 真顔

# 動作について
* win11のPython 3.10.11でのみ確認
* 表情認識は重いのでご承知おきを
* 初期実行時がモデルのDLなどあるので重いです
* 初回実行時にSE用の音声ファイルを効果音ラボから自動DLします
* 終了はターミナルなどから Ctl+C などで強制終了のみ対応
* タイピングが速いと認識できません(Nキーロールオーバーではない)
